# 🚀 Prashant Parmar's Portfolio

Welcome to the repository for my personal developer portfolio, built to showcase my skills, projects, and journey as a Web Developer. This portfolio features a highly interactive and modern UI with smooth animations, dark/light mode support, and a fully functional contact form.

## 🌟 Live Demo

*(Add your live URL here once deployed, e.g., [https://prashant-portfolio.netlify.app](https://prashant-portfolio.netlify.app))*

## ✨ Features

- **Dynamic Loading Screen**: A custom animated loading state to engage users before the main content appears.
- **Dark & Light Mode**: Seamless toggling between beautiful gradient themes, utilizing CSS variables and React state.
- **Scroll Animations**: Smooth reveal animations as you scroll down the page, powered by Intersection Observers and Framer Motion.
- **Project Showcase**: Detailed cards for my projects like **KisanDost**, Coursera Clone, and more, complete with tags, Github, and Live links.
- **Functional Contact Form**: Integrated with EmailJS to send messages directly from the UI without needing a backend.
- **Responsive Design**: Flawless experience across mobile, tablet, and desktop devices.
- **Glassmorphism UI**: Beautiful semi-transparent interactive elements.

## 🛠️ Technologies Used

- **Frontend Framework:** React 19
- **Build Tool:** Vite
- **Styling:** Tailwind CSS v4 + Vanilla CSS Custom Animations
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Email Service:** EmailJS

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   cd portfolio/Portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up EmailJS (Optional)**
   - If you want the contact form to work locally, update the EmailJS `init` public key and template IDs in `src/App.jsx` with your own credentials.

4. **Run the development server**
   ```bash
   npm run dev
   ```

5. **Open in Browser**
   - Head over to `http://localhost:5173` to see the portfolio in action!
